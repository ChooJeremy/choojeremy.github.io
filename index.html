These are the projects I'm currently working on.
Teammates An open source project based on giving feedback or evalutions. Typically used by instructors for students. Used by 250,000 users in 1,773 institutions over 96 countries. Star Pelt A reaction based game where stars come at you and you have to block them. The idea is that as you do so a randomly generated melody will play. You can also use your own soundtrack and have the stars follow that soundtrack.
I consider these projects completed and it is unlikely that I will continue working on them.
Network Intrusion Detection System A network intrusion detection system built to detect malicious traffic over legitimate traffic and block them in realtime, as well as detecting zero-day attacks. Competitve Elevator Competitive Elevator - Can you fax more people than your opponent in an action paced elevator simulator? Available on Android for now. Myo Space Shooter Space Shooter - A 2d arcade-style game that uses the capabilities of the Myo armband to control the player. Created for stroke patients. GRPG Generic Role-Playing Game - A 2d top down open world game similar to RuneScape, created in just 3 weeks and featured in Ngee Ann's showcase.
MeL pentesting MeL Vulnerability - A security vulnerability of Ngee Ann's learning portal was discovered and exposed to the relevant people. The King's Hand The King's Hand - A 3d fps Archery Assassination Stealth Mission based game similar to Assassin's Creed, but uses bows intead of guns. Who Knows Who Knows - A game inspired from Wits and Wages. It disguises learning as a game, in an attempt to help students learn and have fun. Little Strike Little Strike - a 2d top down multiplayer shooter game, using a peer to peer system, featuring multiple customizations for every player.
Mini CTF A simple, minature CTF for users to try out and see if they can complete the challenges. Contains a rule page and shortcuts for all the challenges. Petals around the rose Simple Petals Around the Rose, a game I created based on the game Petals Around the Rose, which challenges players to think outside the box Calculator SNAP A Calculator that I developed for Windows 8, as part of my Microsoft Student Partners program. It features a text calculator to be used snapped. Indian Poker A command-line game that runs a game of Indian Poker, also known as Blind man's bluff, a version of poker where you can't see your card.
